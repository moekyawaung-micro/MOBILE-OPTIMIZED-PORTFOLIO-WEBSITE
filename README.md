# 📱 Mobile-Optimized Portfolio & Tools

Mobile-first responsive portfolio website with advanced copy-paste, file upload, and text manipulation tools.

## 🌟 Features

- ✅ **Fully Responsive** - Works on all devices
- ✅ **Progressive Web App (PWA)** - Install as native app
- ✅ **Offline Support** - Works without internet
- ✅ **File Upload** - Drag & drop, paste support
- ✅ **Copy-Paste Tools** - Text, code, data manipulation
- ✅ **Mobile Optimized** - Touch-friendly interface
- ✅ **Dark Mode** - Easy on the eyes
- ✅ **Fast Loading** - Optimized performance

## 📱 Mobile Features

### Copy & Paste Helper
- Text formatting (uppercase, lowercase, reverse)
- Code beautification (JSON, JavaScript, Python, etc.)
- Data conversion (CSV, JSON)
- Base64 encoding/decoding
- URL encoding/decoding
- Password generation
- Real-time character count

### File Upload
- Drag and drop support
- Clipboard paste support
- File size validation
- Progress tracking
- File preview
- Multiple file support

### Portfolio
- Mobile-first design
- Touch-optimized navigation
- Safe area support (notches, corners)
- Reduced motion support
- High contrast mode support
